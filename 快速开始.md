## 📝 使用步骤

### 方法一：双击批处理文件（最简单）

1. **运行程序**
   ```
   双击: 运行程序.bat
   ```
   - 会自动以管理员身份启动
   - 会弹出UAC提示，点击"是"

2. **加载驱动**
   - 在程序界面点击"加载驱动"按钮
   - 选择编译好的 `.sys` 驱动文件
   - 等待驱动加载成功

3. **隐藏进程**
   - 在进程列表中选择要隐藏的进程
   - 点击"隐藏选中进程"按钮
   - 确认操作

### 方法二：直接运行exe

1. **找到可执行文件**
   ```
   路径: Bin\hipro.exe
   ```

2. **以管理员身份运行**
   - 右键点击 `hipro.exe`
   - 选择"以管理员身份运行"
   - 或按 `Shift + 右键` → "以其他用户身份运行"

## ⚙️ 前置要求

### 1. 禁用驱动签名强制（必须）

在使用驱动之前，需要禁用Windows的驱动签名验证：

```powershell
# 以管理员身份运行PowerShell
bcdedit /set testsigning on

# 重启计算机
shutdown /r /t 0
```

重启后，桌面右下角会显示"测试模式"字样。

### 2. 编译驱动程序（可选）

如果你还没有编译驱动，需要先编译：

**需要安装：**
- Visual Studio 2019/2022
- Windows Driver Kit (WDK)

**编译步骤：**
1. 打开 `hipro.sln`
2. 右键点击 `hipro` 项目
3. 选择"生成"
4. 驱动文件位置: `x64\Release\hipro.sys`

## 🎯 程序功能

### 主界面功能

- **驱动状态显示**: 实时显示驱动是否加载
- **连接状态**: 显示是否与驱动成功通信
- **进程列表**: 显示所有运行中的进程
  - PID（进程ID）
  - 进程名称
  - 完整路径
- **搜索框**: 快速查找进程
- **刷新按钮**: 更新进程列表
- **加载/卸载驱动**: 管理驱动程序
- **隐藏进程**: 将选中进程从系统列表中移除

### 操作流程

```
1. 启动程序（管理员权限）
   ↓
2. 加载驱动（选择.sys文件）
   ↓
3. 查看进程列表
   ↓
4. 选择要隐藏的进程
   ↓
5. 点击"隐藏选中进程"
   ↓
6. 确认操作
   ↓
✓ 完成！进程已隐藏
```

## ⚠️ 重要提示

### 安全注意事项

1. **仅用于学习研究**
   - 不要用于非法用途
   - 遵守当地法律法规

2. **谨慎选择进程**
   - ❌ 不要隐藏系统关键进程（如 csrss.exe, svchost.exe）
   - ✅ 可以隐藏自己开发的测试程序
   - ✅ 可以隐藏游戏进程（仅供学习）

3. **恢复方法**
   - 重启计算机可恢复所有隐藏进程
   - 卸载驱动不会自动恢复进程链接

4. **系统稳定性**
   - 隐藏某些进程可能导致系统不稳定
   - 建议在虚拟机中测试

## 🔧 故障排除

### 问题1：程序无法启动

**症状**: 双击后没有任何反应

**解决**:
- 确保以管理员身份运行
- 检查是否有杀毒软件拦截
- 查看Windows事件日志

### 问题2：无法加载驱动

**症状**: 点击"加载驱动"后报错

**解决**:
```powershell
# 检查是否已禁用驱动签名
bcdedit | findstr testsigning

# 如果不是on，执行：
bcdedit /set testsigning on
shutdown /r /t 0
```

### 问题3：找不到编译的exe

**症状**: Bin文件夹不存在

**解决**:
```
双击运行: 编译客户端.bat
```
